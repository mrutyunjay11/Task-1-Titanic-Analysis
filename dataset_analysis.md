## Dataset Overview
The Titanic dataset contains information about 891 passengers,
including personal details and survival status.

## Feature Types
Numerical features include Age and Fare.
Categorical features include Sex and Embarked.
Passenger class is an ordinal feature.
Survival status is a binary target variable.

## Target Variable
The target variable is Survived, which indicates whether a passenger lived or died.

## Data Quality Issues
Age has around 20% missing values.
Cabin has more than 70% missing values.
Some columns like Name and Ticket are not useful for modeling.

## Machine Learning Suitability
The dataset is suitable for classification tasks.
Basic preprocessing like handling missing values is required.
